Get Started

TiKV provides both raw key-value APIs (RawKV) and transactional APIs (TxnKV) with ACID compliance.

TiKV is an ideal choice in the following scenarios

As an online serving module of a recommendation system or a feature store
As a key-value store, e.g. store metadata for an object storage system
As a storage layer for a database, e.g.
  • TiDB: a distributed MySQL protocol compatible NewSQL Database
  • Zetta: a distributed HBase protocol compatible NoSQL Database
  • Tidis: a distributed Redis protocol compatible NoSQL Database
